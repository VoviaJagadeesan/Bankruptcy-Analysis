#  Bankruptcy Prediction using Machine Learning

## Project Overview
This project focuses on predicting whether a company is likely to go bankrupt based on various risk factors such as industrial risk, management risk, financial flexibility, credibility, competitiveness, and operating risk.

The goal is to build a reliable classification model that can assist in early detection of financial distress and support data-driven decision-making.

##  Objective
- Analyze key factors influencing bankruptcy
- Perform Exploratory Data Analysis (EDA)
- Identify patterns and relationships in the dataset
- Build and evaluate multiple classification models
- Select the most robust and reliable model

##  Exploratory Data Analysis (EDA)
- No missing values detected in the dataset
- Features are ordinal categorical (0, 0.5, 1)
- Data shows discrete distributions with minimal outliers
- Higher risk values are associated with bankruptcy cases
- Moderate correlations exist between some features

## Models Used
- Logistic Regression  
- Support Vector Machine (SVM)  
- Random Forest  
- Gradient Boosting  

## Key Findings
- Dataset is clean with no missing values  
- Risk-related features strongly influence bankruptcy  
- Higher risk levels increase the probability of bankruptcy  
- Models achieved high accuracy due to clear data patterns  
- Cross-validation was performed to ensure model reliability  

##  Business Insights
- Early identification of high-risk companies is possible  
- Financial flexibility plays a critical role in survival  
- Credibility and management efficiency impact stability  
- Predictive models can support investment and lending decisions  

##  Conclusion
The project successfully demonstrates that machine learning models can effectively predict bankruptcy. Among the models tested, Random Forest was selected as the final model due to its robustness, interpretability, and consistent performance.
